I was always fascinated with programming and Mobile environments but never got around doing that. So, since then, I have been learning how to code and have gotten familiar with various parts of the Front end and Back end side of the development. I recently finished a course of Web & Mobile developer, which includes HTML5, CSS3, JavaScript and the Framework Bootstrap; Multi-Platform Mobile Application Development using Cordova, Ionic, Node.js, AngularJS; CMS of which Drupal and WordPress; Native Java Development for Android, SEO / SEM and more recently I’m finishing to learn AngularJS with the Ionic Framework for mobile development.


I’m a conscientious and professional person, with extremely high standards and eye for details, currently seeking a position that will make use of acquired skills with and opportunity for growth. A highly methodical and efficient individual, whose thorough and precise approach to work has yielded excellent results.
